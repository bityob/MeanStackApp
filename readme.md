# MEAN Stack- Supermarket app 		

 Complete MEAN stack application with authentication using json web tokens		
		
 ## Usage		
	
 ### Installation		
 		
 Install the dependencies		
 		
 ```sh		
 $ npm install		
 ```		
 	
 Server start:		
 ```sh		
 $ nodemon		
 ```		
	
 Client start:		
 ```sh		
 $ ng serve		
 ```		

Install MongoDb:
```bash
$ sudo apt-get install -y mongodb-org
```
